Paste this block at the start of any new TruSkills session.

---

## Context Load Instructions

1. Load `00_Master_Continuity_Record.md` now as the active locked context for TruSkills.

2. Follow the Memory Use Triggers defined in that file:
   - If the first task is clearly Courses work → also load `01_Courses_Memory_Pack.md`
   - If the first task is clearly build, deployment, governance, or sprint-end audit work → also load `02_Build_Governance_Memory_Pack.md`
   - If the task is not yet known → stay on master context only until the task is given

3. Check the **Current Next Step** section in `00_Master_Continuity_Record.md` and use it to identify which scoped pack (if any) should be pre-loaded before I give my first task.

## Behaviour Rules

- Do not ask me to restate the project, the platform, or any locked decisions.
- Do not reopen any decision listed in the Do Not Reopen section of the master record.
- Do not load unnecessary memory packs.
- Do not summarise the files back to me after loading.
- After loading, report current position, propose the next step, and wait for my confirmation before proceeding.

## Confirmation Format

After loading, respond with exactly this structure:

> **Context loaded.** [Pack(s) loaded: list or "Master only"]
>
> **Current position:** [state the active workstream or "No active workstream" as recorded in Current Next Step]
>
> **Proposed next step:** [state the next recommended action based on Current Next Step]
>
> Confirm to proceed, or give me a different task.
